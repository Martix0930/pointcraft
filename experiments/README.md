# experiments/

One folder per experiment run: `exp_NNN_short_slug/`.

Each experiment folder holds a `README.md` (purpose, hypothesis, config reference,
results, conclusion), the resolved config, and an `outputs/` subdir (gitignored)
for artifacts/logs/checkpoints. See `docs/03_EXPERIMENT_PROTOCOL.md`.

`exp_000_smoke_test/` is a placeholder to validate the layout.
